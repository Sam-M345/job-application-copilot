# Resume Formatting Rules

Based on direct comparison of a well-formatted resume vs. a plain one.

## Page Setup
- US Letter (8.5" x 11")
- Margins: 0.75" left/right, 0.625" top/bottom

## Fonts
- Body font: Arial, throughout (set explicitly, never leave Word's default theme font)
- Name/header: 16pt, bold, blue accent color `#1F5C99`
- Section headings (SUMMARY, EXPERIENCE, SKILLS, etc.): 12pt, bold, blue accent `#1F5C99`, ALL CAPS
- Body text and bullets: 11-12pt, black
- Secondary/muted text (e.g. locations): dark gray `#444444`

## Header Block
- Name, contact line, hyperlinks (LinkedIn/Portfolio/Kaggle), certifications line: all center-aligned
- Contact/links in the blue accent color to visually separate from body
- Rest of the document left-aligned

## Section Headers
- Bold, blue, ALL CAPS
- Thin bottom border/rule under each section heading (single line, 0.75pt weight, blue `#1F5C99`)
- Consistent spacing before (~140 twips) and after (~20 twips) each heading

## Job Entries
- Company/title in bold
- Dates right-aligned on the same line as the job title/company, using a right tab stop (not spaces, not a separate line)
- Bullets use consistent paragraph spacing (~20 twips before/after)

## Bullets
- Use real bullet list formatting (numbering/list style), never typed dashes or manual bullet characters
- Bold key metrics/tools within bullets sparingly, only where it earns emphasis

## What to Avoid
- Unstyled default theme fonts (Calibri/Cambria left untouched)
- Missing color hierarchy (all-black text with only bold for distinction reads flat)
- Oversized margins (1"+ wastes page space and pushes content to extra pages)
- Inline typed dates without tab alignment
- Dense unbroken paragraphs without section-divider borders
